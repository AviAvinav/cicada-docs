---

Set up autocomplete/intellisense for the Cicada TypeScript SDK

### 1. Install the Deno extension for VSCode:

```bash
code --install-extension denoland.vscode-deno
```

If you don't have `code` installed, then click [here](https://marketplace.visualstudio.com/items?itemName=denoland.vscode-deno) to install the extension

### 2. Add the following to your .vscode/settings.json

```json
{
  "deno.enablePaths": [".cicada"]
}
```

You will have to restart VSCode in order for the settings to take effect
