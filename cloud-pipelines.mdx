---
title: Run pipelines in the cloud
description: "Connect Cicada to a GitHub repository"
---


### 1. Sign up to Cicada

Sign up at [cicada.build](https://cicada.build/dashboard)

### 2. Link your GitHub repository

Choose the GitHub repository you want to connect. 

### 3. That's it!
Now whenever you / your teammates create a PR, Cicada will comment on it with the status of your pipeline. You'll be able to see your logs at [cicada.build](https://cicada.build/dashboard)
