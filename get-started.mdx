## 0. Dependencies
Cicada depends on [Docker](https://docs.docker.com/engine/install/) & [deno](https://deno.land/manual@v1.32.3/getting_started/installation) installed


## 1. Install Cicada CLI


<Tabs>
  <Tab title="yarn">
    
    ```bash
    yarn add -D @cicadahq/cicada
    ```

  </Tab>
  <Tab title="npm">
    
    ```bash
    npm install -D @cicadahq/cicada 
    ```

  </Tab>
  <Tab title="brew">
    ```bash
    brew install cicada
    ```
  </Tab>
  <Tab title="curl">
    ```bash
    curl -fSsL https://raw.githubusercontent.com/cicadahq/release/main/download.sh | sh
    ```

  </Tab>
</Tabs>



## 2. Create a new pipeline
Navigate to a project then run 
```bash
cicada init
```
This will create an example pipeline at `.cicada/my-pipeline.ts`


## 3. Run your pipeline
```bash
cicada run my-pipeline
```

That's it! Now set up your pipeline so it runs in the cloud
